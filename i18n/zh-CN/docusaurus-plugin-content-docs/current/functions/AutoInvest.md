---
title: 省心投
---



为方便您使用 ExinOne 进行自动定投，ExinOne 推出了“省心投”功能：

您只需要将您投资用的稳定币存入「活期宝」或者「省心投专属账户」内，即可自动定投。

您可以提前转入资产至【省心投专属账户】，当您的定投组合打开【省心投】开关后，每期定投费用默认在【省心投专属账户】中扣除相应的 USDT，USDC 或 pUSD。

当【省心投专属账户】中的余额不足时，系统会发起从【活期宝账户】中扣除相应的 USDT，USDC 或 pUSD，不会同时从 2 个账户中扣款。

【省心投专属账户】

平台将会为每个用户创建一个基于 Mixin Network 的省心投专属钱包，用户可选择将 USDT，USDC 或 pUSD 存入该钱包进行自动定投；

省心投专属账户将会提供持仓账户的访问令牌，用户可以通过该令牌在 Mixin Network 浏览器中实时查询当前的持仓，该账户的资产除省心投功能自动扣款外，不会被挪用；

省心投专属账户的资产，用户可以随时存取，不会锁仓，没有收益。

温馨提示：

当省心投专属账户或活期宝账户中 USDT，USDC 或 pUSD 金额充足时，抵扣成功，定投完成，自动执行；

当省心投专属账户或活期宝账户中 USDT，USDC 或 pUSD 金额不足时，需要用户手动支付，系统会发送定投订单生成，请及时支付的信息给用户，同时，定投页面对应的定投组合下面会显示【手动支付】按钮；
